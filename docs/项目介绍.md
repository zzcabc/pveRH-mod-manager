# 重构mod管理器

## 页面配置
- 第一行 `游戏目录：选择框(允许切换)` `版本：选择框`
- 第二行 `MOD目录：选择框(允许切换)`
- 第三行 `BepInEx检测：已安装/未安装` `安装BepInEx(未安装)/卸载BepInEx(已安装)`
- 第四行 `安装高数修改器` `安装梧萱修改器`
- 第五行 `打开游戏目录` `打开MOD目录` `打开已安装目录`
- 第六行 `植物MOD(未安装)` `植物MOD(已安装)`
- 第七行 `僵尸MOD(未安装)` `僵尸MOD(已安装)`
- 第八行 `皮肤(未安装)` `皮肤(已安装)`
- 第九行 `在线MOD`

## 基础配置

- 在程序执行目录下生成 `config.json`内容如下，游戏目录和MOD目录允许选择多个，版本根据MOD目录内版本进行选择

```json
{
  "game_path": [
    {
      "path": "E:\\Game\\植物大战僵尸融合版3.7Mod",
      "version": "3.7"
    },
    {
      "path": "E:\\Game\\植物大战僵尸融合版3.6.1Mod",
      "version": "3.6.1"
    }
  ],
  "mod_path": [
    "E:\\Game\\MOD"
  ],
  "download_path": ""
}
```
- `download_path`是下载MOD目录

## 添加检测`BepInEx`

- 读取项目内的`gamefile.json`中`BepInEx`内容 , 查询游戏目录下有没有这些内容，如果存在则为已安装，否则为未安装
- 点击`安装BepInEx`查询`BepInEx.zip` 解压到游戏目录下
- 点击`移除BepInEx`保留`gamefile.json` 中`GameFile`内容，移除其他文件或文件夹

## 添加安装修改器 

点击安装`安装高数修改器`, 检查mod文件夹有没有 类似`PvZRHModfiedFor(版本)[].zip`格式的压缩包，记得匹配版本，如果有解压到游戏目录下，如果没有查询网站这个文件(查询功能暂时不做，等待服务器程序)

## MOD管理
1. 未安装mod为mod文件夹内的mod，显示名取mod所在目录名
2. 已安装mod为游戏目录下`BepInEx/plugins`文件夹内的mod，显示名取mod所在目录名
3. 点击`移除已安装mod`，移除游戏目录下`BepInEx/plugins`文件夹内的mod
4. 僵尸同上
